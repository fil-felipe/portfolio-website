Gift list

The application is built using the Python Django framework. It requires users to log in, but provides the convenience of logging in through Facebook, Google, or Twitter. Users can create their own wish lists, as well as create lists for others. Additionally, there's an option to share wish lists with other users, who can then reserve gifts. Reserved gifts automatically disappear from the list. Furthermore, adding a new gift involves web scraping, where based on the URL of the gift offer, the application retrieves the image URL and gift title.

Application use PostgreSQL as database. Production version works on my own hardware which use linux server to host websites.
